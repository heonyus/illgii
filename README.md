# 일기 웹사이트

브런치, 미디엄, 쓰레드의 장점을 결합한 심플한 일기 웹사이트입니다.

## 특징

- 🎨 토스 스타일의 미니멀한 디자인
- ✍️ 마크다운 편집 지원
- 📱 모바일/웹 최적화
- 🎭 부드러운 애니메이션 효과
- 💾 로컬스토리지 기반 데이터 저장

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Markdown
- Framer Motion

## 시작하기

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 프로덕션 빌드

```bash
npm run build
npm start
```

## 배포

Vercel에 배포하는 방법:

1. [Vercel](https://vercel.com)에 로그인
2. 새 프로젝트 생성
3. GitHub 저장소 연결 (또는 직접 배포)
4. 자동으로 빌드 및 배포됩니다

## 사용법

1. 메인 페이지에서 "새 일기" 버튼 클릭
2. 제목과 마크다운 형식으로 일기 작성
3. 저장 버튼으로 저장
4. 목록에서 일기를 클릭하여 읽기
5. 편집 또는 삭제 가능
